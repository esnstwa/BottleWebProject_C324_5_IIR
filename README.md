# Кинематика: интерактивное веб-приложение для моделирования физических задач

## О проекте

Данный проект разработан в рамках учебной практики **УП02**.

Приложение представляет собой веб-инструмент для численного решения и визуализации трёх задач кинематики:
1. **Сложное движение** (переправа катера через реку с учётом течения)
2. **Графики движения** (вертикальное движение тела в поле силы тяжести)
3. **Относительность движения** (встречное движение двух протяжённых объектов)

Пользователь может изменять исходные параметры моделей, выполнять расчёты и наблюдать результаты в числовом и графическом виде через браузер.

## Основной функционал

- Главная страница с навигацией по трём задачам и теоретическим сведениям.
- Страница «Об авторах» с информацией о вкладе каждого участника.
- Интерактивные формы ввода параметров с валидацией.
- Построение графиков (для задачи «Графики движения»).
- Визуализация процессов (анимация или схемы для задач переправы и встречного движения).
- Сохранение входных данных и результатов в JSON-файл (`calculations_history.json`).
- Модульное тестирование расчётных функций (библиотека `unittest`).

## Технологический стек

- **Язык:** Python 3.x
- **Веб-фреймворк:** Bottle
- **Тестирование:** unittest (стандартная библиотека)
- **Формат данных:** JSON
- **Система контроля версий:** Git + удалённый репозиторий (GitHub)
- **Среда разработки:** Microsoft Visual Studio Community 2019 (или любой редактор с поддержкой Python)

## Инструкция по установке и запуску

### Требования к системе

- **Операционная система:** Windows 10/11, macOS или Linux
- **Python:** версия 3.8 или выше
- **Браузер:** Google Chrome, Mozilla Firefox или любой современный браузер
- **Система контроля версий:** Git (опционально, для клонирования репозитория)

### Способ 1: Клонирование через Git (рекомендуется)

```bash
# 1. Клонировать репозиторий
git clone https://github.com/ваш_username/kinematics-lab.git

# 2. Перейти в директорию проекта
cd kinematics-lab

### Способ 2: Скачать ZIP-архивом

1. Перейти на страницу репозитория GitHub
2. Нажать кнопку "Code" → "Download ZIP"
3. Распаковать архив в удобную директорию

Установка зависимостей
```bash
# 1. Создать виртуальное окружение (рекомендуется)
python -m venv venv

# 2. Активировать виртуальное окружение

# Windows:
venv\Scripts\activate

# macOS / Linux:
source venv/bin/activate

# 3. Установить необходимые библиотеки
pip install -r requirements.txt
```
Запуск приложения:
```bash
# 1. Убедиться, что виртуальное окружение активировано
# (в терминале должно быть (venv) в начале строки)

# 2. Запустить сервер
python app.py

# 3. Увидеть сообщение об успешном запуске:
# ------------------------------------------------------------
# Bottle server starting up (using WSGIRefServer())...
# Listening on http://localhost:8080/
# Hit Ctrl-C to quit.
# ------------------------------------------------------------
```
